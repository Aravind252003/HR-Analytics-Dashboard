# HR-Analytics-Dashboard

## Project Overview
This HR Analytics Dashboard helps analyze employee attrition, salary trends, age distribution, and department-wise performance.

The dashboard was built using Power BI to generate insights useful for HR decision-making.

## Key Metrics
- Total Employees: 1470
- Attrition Count: 237
- Attrition Rate: 16.12%
- Average Age: 36.92
- Average Salary: 6.50K

## Dashboard Insights
### Attrition by Education Field
- Life Sciences has the highest attrition.
- Medical field employees also show significant attrition.

### Attrition by Job Role
- Sales Executive has maximum attrition.
- Research Scientist and Laboratory Technician follow next.

### Attrition by Age
- Employees aged 25–34 show highest attrition.
- Attrition decreases with higher age groups.

### Salary Hike Analysis
- Employees with lower salary hikes tend to leave more.
